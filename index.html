What is ReducedJSON?

ReducedJSON is JavaScript library to take all the responsibility to parse the JSON data structure at run time dynamically, so you do not need to write complex parsing code for any length and complexity level of JSON data. Make your development pace 75% faster to concentrate on business feature development rather than consuming much of the time and efforts in writing parsing code to parse the JSON data

ReducedJSON library can be used with any JavaScript framework. It is quite efficient and effective while playing with JSON data to parse and find target data. There is need of very basic level understanding of JavaScript to use ReducedJSON library. Soon, ReducedJSON will help jQuery framework users to come back into the race with AngularJS and React JavaScript frameworks users.

How to use it?

It is very handy to use ReducedJSON in quickest and shortest matter of time like in minutes. One needs to have good knowledge of Array and Map data structure while understanding JSON data

There are two important parameters will always be used in ReducedJSON library while calling it's reduce(json,command) function.

In reduce(json,command) function, json is a string of json data and nothing more than that. where as command is what all about to understand how ReducedJSON works
